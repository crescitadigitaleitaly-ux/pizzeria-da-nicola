# Ristorante Pizzeria Da Nicola — Savona

Sito web del Ristorante Pizzeria Da Nicola (Savona, dal 1956): cucina ligure e campana, pizza napoletana, pesce fresco.

## Funzionalità
- Menu completo con foto, descrizioni e prezzi
- Ordine online (asporto / consegna) con carrello
- Pagamento contanti o carta online
- Prenotazione tavolo online
- Galleria foto del locale
- Mappa e indicazioni stradali
- Italiano, mobile responsive

## Struttura
- `index.html` — sito completo (HTML/CSS/JS, nessuna build)
- `assets/img/` — foto del locale, dei piatti e mappa

## Avvio locale
Apri `index.html` nel browser, oppure servi la cartella con un web server statico.
